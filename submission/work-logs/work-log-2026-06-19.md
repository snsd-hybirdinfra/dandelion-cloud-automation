## 7. 작업 결과 요약

2026년 6월 19일 작업에서는 Phase 1 기본 구성과 운영 확장 초기 구성이 함께 진행되었다.

백서빈은 디스크 용량을 300 GB 기준으로 증설하고, 리소스표 기준 인스턴스 생성 및 SSH 접속 검증을 완료하였다.

이진욱은 /etc/hosts 작성, SSH config 설정, 각 노드별 공통 패키지 설치, DB Node MariaDB 설치 및 설정, DB 생성, DB 계정 생성을 완료하였다.

조민석은 Control Node에 Ansible을 설치하고 ansible.cfg, inventory.ini, site.yml, common.yml을 작성하였다. 또한 managed 그룹 대상 Ansible Ping과 sudo 권한 상승 검증을 완료하였으며, common.yml 문법 검사까지 완료하였다. 실제 common.yml 실행은 다른 팀원의 패키지 설치 작업과 충돌 방지를 위해 보류하였다.

박재우는 Prometheus 설치, 운영 스크립트 작성, cron 등록을 완료하였다.

금일 작업을 통해 인스턴스 생성 이후 서버 기본 구성, DB 초기 구성, Ansible 중앙 관리 구조, Monitoring 초기 구성까지 진행되었으며, 다음 단계에서는 Ansible Playbook 실제 실행, DB 연동 검증, Prometheus target 확인, cron 실행 로그 확인을 진행할 예정이다.
